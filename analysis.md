### 매칭 상세

**EZ-Approve (1순위)**
- RBAC 7종 역할 관리 → CS CRM 3종 역할(슈퍼관리자, CS관리자, CS상담사)에 직접 활용
- 12개 외부 서비스 연동 패턴 → 채널톡/하우저 API 연동에 재활용
- B2B 관리자 대시보드 구조 → CRM 대시보드 아키텍처 동일

**Harmony Link (2순위)**
- 이용자(고객) 이력 통합 관리 → 상담 이력 + 주문 이력 통합 뷰와 직접 유사
- 실시간 푸시 알림(FCM) → 채널톡 웹훅 실시간 알림과 패턴 유사
- NestJS + Next.js + TypeORM 동일 스택

**Series B (3순위)**
- 80+ 엔티티 규모 데이터 모델링 → 안정적 DB 설계 경험 증명
- 다중 API 연동(Firebase, SMS, NICE, ChatGPT) → 외부 시스템 통합 역량
- 179개 마이그레이션 → 스키마 진화 관리 역량
